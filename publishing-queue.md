## How This Works
1. Every Tuesday at 8 AM a GitHub Action automatically opens a PR for the next article
2. Open the PR → click the article link in the description to read the fully rendered markdown
3. **To approve:** Merge the PR
4. **To request changes:** Leave a comment on the PR, then open Claude Code and say `revise [article title]: [your feedback]`
5. **To publish in Squarespace after merging:**
   - Open the article file in the repo → click **Raw** → copy all

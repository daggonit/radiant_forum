# Barrier Boss — Article Topic Queue

## How This Works
Each Tuesday, the GitHub Action picks the next `pending` topic, generates a full article using Claude AI, and opens a PR for your review.

- **To approve:** Merge the PR
- **To request changes:** Leave a comment on the PR, then ask Claude to revise it
- **To skip a topic:** Change its status to `skipped`
- **To add new topics:** Add a row with `pending` status
